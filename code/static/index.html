How it works

  1. Select an analysis above
  2. Click Open in Napari — napari opens with all images from data/input/
  3. Browse with ← → keys, press Space to mark, S to save
  4. Close napari, then click Copy Selections
  5. Analysis Set → data/curated/<analysis>/<strain>/ — used for measurement
  6. Training Set → data/training/<strain>/ — draw masks here, then train a model
Supported formats: .tif  .tiff  .TIF  .TIFF  .jpg  .jpeg  .JPG  .JPEG  .png  .PNG  ·  OME-TIFF pixel size read from metadata automatically
